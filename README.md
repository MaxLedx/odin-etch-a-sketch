# odin-etch-a-sketch

## Description

Etch-a-Sketch is a simple web-based drawing application that allows users to create pixel art by interacting with a grid of squares. Users can hover over the squares to change their color, effectively "drawing" on the canvas. The application features a clear button to reset the grid and a resize button that allows users to change the grid size dynamically, providing a customizable drawing experience. Built with HTML, CSS, and JavaScript, this project serves as a fun introduction to DOM manipulation and event handling in web development.
